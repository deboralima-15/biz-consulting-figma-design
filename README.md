# Biz Consulting Website Design

A responsive business website prototype designed in Figma for a fictional accounting and consulting company based in Perth.

This academic project focused on creating a professional and user-friendly interface for desktop and mobile devices.

## Project Overview

The design includes:

- Home page
- About page
- Services page
- Tax Returns page
- Contact page
- Business Services page
- Business Compliance page
- Business Development page
- Payroll and Accounting page
- Business Broking page

## Tools and Skills

- Figma
- UI Design
- UX Design
- Responsive Design
- Wireframing
- Interactive Prototyping
- Desktop and Mobile Layouts
